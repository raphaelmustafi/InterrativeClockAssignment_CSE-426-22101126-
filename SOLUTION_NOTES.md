# CSE 426 Lab Assignment Solution(22101126, SEC: C-2, 4-2)
## Covered Requirements

- Clock face: drawn with the midpoint circle algorithm using multiple concentric circles.
- Hour markings: drawn around the face with Bresenham raster lines.
- Clock hands: hour, minute, and second hands use Bresenham's line drawing algorithm.
- Thickness: each hand has a different configurable thickness; the second hand is thinner than the hour hand.
- Orientation: the Bresenham implementation handles all octants by using signed x/y steps.
- Animation: GLUT timer callback updates the hands continuously using current time or accelerated demo speed.
- Interaction: keyboard and right-click menu controls are provided for pause/resume, reset to current time, speed, radius, hand widths, theme, and clipping mode.
- Clipping: circular mask clipping and Cohen-Sutherland rectangular line clipping are both included.

## Main Algorithms

### Midpoint Circle Algorithm

The clock rings are generated by starting at `(0, r)`, maintaining a decision parameter, and plotting the eight symmetric points around the center for each step.

### Bresenham Line Algorithm

Every hand and tick mark is rasterized by an integer error accumulator. The implementation supports all line directions by using `sx`, `sy`, `dx`, and `dy` signs.

### Thickness

For a thick raster line, each Bresenham point stamps a small filled disk. This keeps the hand visually continuous while still depending on the Bresenham path.

### Cohen-Sutherland Clipping

The rectangular clipping mode assigns each endpoint a region code and repeatedly intersects the line against the active boundary until the segment is accepted or rejected.

## File

## Controls

- `P`: pause/resume
- `N`: reset to current system time
- `+` / `-`: increase/decrease animation speed
- `T`: change theme
- `C`: cycle clipping mode
- `[` / `]`: decrease/increase clock radius
- `1`: change hour hand thickness
- `2`: change minute hand thickness
- `3`: change second hand thickness
- Right click: open menu

## Compile Examples

Windows with MinGW/freeglut:

```bash
g++ InteractiveClock.cpp -o clock.exe -lfreeglut -lopengl32 -lglu32
```

Linux:

```bash
g++ InteractiveClock.cpp -o clock -lglut -lGL -lGLU
```
